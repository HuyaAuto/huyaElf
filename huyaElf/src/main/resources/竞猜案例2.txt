r1 = x
r2 = y

压豆数1 num1 = a
压豆数2 num2 = b

得到豆子1 get1 = a + a/x
得到豆子2 get2 = b + b/y

赢得豆子数 a/x - b > 0

赢得豆子数 b/y - a > 0

不论输赢，赢得豆子数相同 a/x - b = b/y - a 

a*(x+1)/x = b*(y+1)/y

b = a*(x+1)*y/((y+1)*x)
a = b*(y+1)*x/(y*(x+1))


利润率 = (1/x-(x+1)*y/((y+1)*x))/(1+(x+1)*y/((y+1)*x))

例如：
x = 0.2
y = 3

a = 10000
b = 45000
